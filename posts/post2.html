New domain name: senyizhang.com

Published at September 7, 2024, 17:35 CST

Although it's ok to let Github run my website, I don't think it's 'cool' enough: ). So I registered domain name senyizhang.com this morning ($10 each year, acceptable for a .com domain) and arranged the cloud server. It's good news that Cloudflare provides free services for websites that don't update frequently and have low data transfer volume while Alibaba and Tencent charge hundreds or thousands of yuan each year. The arranging process was quite easy. Just authorizing Cloudflare to visit the Github repository and the website will be online within a few hours.
